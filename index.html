The purpose of this example is to demonstrate the simple development workflow enabled by Evothings Workbench.

  1. Press the "Code" button in the Workbench window to see the source file of this example app.
  2. Open the file "index.html" in a text editor.
  3. Make some changes and save the file.
  4. Voilà! If everything works, the app automatically reloads!
